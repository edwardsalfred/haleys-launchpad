# Story video goes here

The landing page (`index.html`) looks for:

```
video/haley-story.mp4     ← the story video itself
video/poster.jpg          ← optional thumbnail shown before playback
```

Drop the file in with those exact names and reload — no code changes needed.
Until then, the page shows a friendly "coming soon" card in place of the
video, and the letter underneath still tells the story in text.

**Recommended format:** MP4 (H.264), landscape 16:9 (1280×720 or larger),
60–90 seconds, under ~50MB if you want fast mobile loading. A phone-recorded
video in good light with clear audio will outperform a polished studio video
for this — the story is the whole point.

See [`docs/haley-video-script.md`](../docs/haley-video-script.md) for a
ready-to-read script with timing.
